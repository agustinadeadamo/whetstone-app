# Architecture Decision Records

This folder contains the architectural decisions made for this project,
in the lightweight ADR format.

Each file captures a single decision: the context, the options
considered, the choice made, and its consequences. ADRs are immutable
once accepted — if a decision is reversed, a new ADR supersedes the
old one with a `Status: Superseded by NNNN` note.

File naming: `NNNN-short-slug.md`, numbered sequentially.
